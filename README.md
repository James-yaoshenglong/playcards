# 中文版 
扑克牌游戏跑得快的python实现<br>
具备局域网联机功能（python网络编程socket模块）<br>
支持局域网内多局游戏同时进行（python多线程并发编程thread模块）<br>
目前还差pygame图形化和异常处理问题未完成
